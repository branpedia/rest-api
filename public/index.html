Dokumentasi API MediaFire Downloader

Pengenalan

API MediaFire Downloader memungkinkan Anda untuk mengekstrak link download langsung dari MediaFire. API ini berguna untuk mengintegrasikan fungsi download MediaFire ke dalam aplikasi Anda.

Authentication

Untuk saat ini, API ini tidak memerlukan authentication. Namun, untuk penggunaan komersial atau volume tinggi, API key mungkin diperlukan di masa depan.

Endpoint

GET /api/mediafire?url={mediafire_url}

Parameter

Parameter yang diperlukan:

  • url (string) - URL MediaFire yang ingin diambil informasi downloadnya

Contoh Penggunaan

Contoh penggunaan dengan curl:

curl -X 'GET' \ 'https://your-vercel-app.vercel.app/api/mediafire?url=https%3A%2F%2Fwww.mediafire.com%2Ffile%2Fvj3al1c98u2zdr6%2FTerakomari_-_MD.zip%2Ffile' \ -H 'accept: application/json'

Response

Response JSON akan berisi:

Response
{ "success": true, "data": { "name": "Terakomari - MD.zip", "size": "6.78MB", "extension": "zip", "uploaded": "2024-02-22 07:29:23", "downloadUrl": "https://download2264.mediafire.com/.../Terakomari+-+MD.zip" } }

Error Handling

Jika terjadi error, response akan mengembalikan success false dan pesan error:

Error Response
{ "success": false, "error": "Invalid MediaFire URL" }

Dokumentasi API Instagram Downloader

Pengenalan

API Instagram Downloader memungkinkan Anda untuk mengunduh konten dari Instagram seperti foto, video, dan IGTV. API ini berguna untuk mengintegrasikan fungsi download Instagram ke dalam aplikasi Anda.

Authentication

Untuk saat ini, API ini tidak memerlukan authentication. Namun, untuk penggunaan komersial atau volume tinggi, API key mungkin diperlukan di masa depan.

Endpoint

GET /api/instagram?url={instagram_url}

Parameter

Parameter yang diperlukan:

  • url (string) - URL Instagram yang ingin diambil kontennya

Contoh Penggunaan

Contoh penggunaan dengan curl:

curl -X 'GET' \ 'https://your-vercel-app.vercel.app/api/instagram?url=https%3A%2F%2Fwww.instagram.com%2Fp%2FCs6Y8-0uL7h%2F' \ -H 'accept: application/json'

Response

Response JSON akan berisi:

Response
{ "success": true, "data": { "type": "video", "caption": "Caption dari postingan Instagram", "downloadUrl": "https://api.branpedia.com/download/instagram/...", "thumbnail": "https://api.branpedia.com/thumbnail/instagram/...", "duration": "0:15", "size": "3.2MB" } }

Error Handling

Jika terjadi error, response akan mengembalikan success false dan pesan error:

Error Response
{ "success": false, "error": "Invalid Instagram URL" }

Tentang Branpedia API

Branpedia API adalah kumpulan layanan API gratis yang dikembangkan untuk memudahkan pengembang dalam mengintegrasikan berbagai fungsi ke dalam aplikasi mereka.

Fitur

  • API Downloader (MediaFire, YouTube, Instagram, TikTok)
  • Layanan AI (Image to Anime, ChatGPT, dll)
  • Tools (URL Shortener, Image Processing, dll)
  • Tanpa batas penggunaan
  • Response cepat dan reliable
